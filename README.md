## 🛠 Tech Stack
This project is built using modern web technologies:
- [Next.js](https://nextjs.org) (App Router)
- [React](https://react.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- Custom CSS Animations & Glitch Effects
- [React Markdown](https://github.com/remarkjs/react-markdown) (for Rules & Regulations)

## 🚀 Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the site in action.

## 📁 Project Structure
- `app/` - Next.js App Router pages and global CSS.
- `components/` - Reusable UI components (GameCards, CTA, Modals, etc.).
- `lib/` - Configuration data and markdown constants (`site.ts`, `rules.ts`).
- `public/` - Static assets like images and SVG backgrounds.

## 📜 Rules & Regulations
Participants must accept the strict Terms & Conditions modal before being redirected to the registration portal. Full rules can be updated via the `lib/rules.ts` file.
